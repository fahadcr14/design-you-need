## Design Recipes

### Score 8/10 — retro-modern — Illustrated icon/badge set — Paris landmark line art

**Design Recipe:** Draw architectural subjects with uniform 2-3px strokes in deep navy #04065D on white. Frame each in a circle with 2px border. Create color variants using #294DBD for structural elements, #D696AE for facades, #E84B5A sparingly for focal points, #7BCAB0 for organic elements. Add halftone dot patterns (4px dots, 8px spacing) to roofs and foliage for texture. Scatter atmospheric dashes and cloud shapes in the sky. Keep all elements geometric and clean — no hand-drawn wobble.

**Dev Recipe:** Export as optimized SVGs with embedded pattern-fills for halftone textures. Use clipPath circles for framing. If implementing as web components, use inline SVG with CSS custom properties for the color swap between mono and color variants — toggle fill values via a single class change.

**Core Lesson:** A constrained color palette (navy #04065D + blue #294DBD + pink #D696AE + one red accent) applied to meticulous line work creates more visual impact than complex coloring ever could

**Steal These:**
- Dual monochrome/color variant system for the same illustration
- Halftone dot-pattern fills for implied texture without gradients
- Circular framing with elements bleeding past the border
- Navy-not-black line work for sophistication
- Atmospheric scatter elements (dashes, clouds, birds) to fill negative space without clutter

**Weaknesses:** Limited to circular format — may not adapt well to rectangular UI contexts, The color palette leans feminine — may not suit all brand contexts, Detail density means these won't read well below ~80px diameter
